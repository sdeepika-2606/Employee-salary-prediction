# 💼 Employee Salary Predictor – Streamlit App

This project is a simple machine learning web app that predicts employee salary based on years of experience using Linear Regression.

## 🧠 Features
- Predict salary instantly
- Built with Streamlit
- Trained using real-world data

## 🚀 How to Use
1. Enter years of experience
2. Click "Predict Salary"
3. Get instant results!

## 🛠 Built With
- Python
- Scikit-learn
- Streamlit
- Pandas
- Joblib
